Summary: Weather vs. Latitude Analysis

This project was designed to compare various weather factors (temperature, wind, humidity and cloudiness) to the latitude in a given location to determine if those weather factors are affected by the locations distance from the equator. To complete this project, we pulled weather date from the OpenWeatherMap API to assemple a dataset including over 500 cities. Once the data was returned, we used Matplotlib to plot weather factors against latitude. This site includes all plots and findings from this analysis as well as the source data.
